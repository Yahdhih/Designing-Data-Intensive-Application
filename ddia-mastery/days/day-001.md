> ⏱ 60 min  |  🔧 Setup 50 · 🧠 Consolidation 10

## 🔧 Setup & Calibration — 50 min

### Étape 1 — Déclare ton édition (5 min)

Ouvre `setup/00-setup.md`. Coche ton édition et note-la :

- **1ʳᵉ éd. (2017) :** 12 chapitres, 3 parties (Partie I : Ch.1-4 · Partie II : Ch.5-9 · Partie III : Ch.10-12)
- **2ᵉ éd. (2026) :** structure réorganisée — à mapper via `reference/toc.md`

### Étape 2 — Calibration de la table des matières (15 min)

Ouvre ton livre. Remplis `reference/toc.md` colonne par colonne :
- Numéro de chapitre · Titre exact · Page de début · Page de fin · Nombre de pages

**Pourquoi c'est crucial :** tous les `days/day-NNN.md` utilisent des `p. ___` qui renvoient à ta TOC. Sans cette calibration, les estimations de lecture sont approximatives.

Pendant que tu remplis, observe : quel est le chapitre le plus long ? (Spoiler : probablement Ch.7 Transactions ou Ch.9 Consensus.)

### Étape 3 — Environnement Python (15 min)

```bash
# Depuis la racine de ddia-mastery/
python --version       # >= 3.10 recommandé
python -m venv .venv
source .venv/bin/activate

pip install -r code/common/requirements.txt

# Vérification
python -c "import psycopg, redis; print('✓ Python OK')"
```

Si une dépendance échoue : `pip install psycopg[binary] redis` séparément.

### Étape 4 — Docker & test du stack léger (10 min)

```bash
# Vérifier Docker
docker --version
docker compose version

# Lancer le stack léger (Postgres + Redis)
docker compose -f labs/compose.light.yml up -d

# Attendre ~10s puis vérifier
docker compose -f labs/compose.light.yml ps
docker compose -f labs/compose.light.yml exec postgres \
    psql -U ddia -c "SELECT version();"

# Arrêter (tu n'en as pas besoin aujourd'hui)
docker compose -f labs/compose.light.yml down
```

**Si Postgres ne démarre pas :** vérifie que le port 5432 n'est pas déjà utilisé (`lsof -i :5432`).

### Étape 5 — Import Anki (5 min)

Ouvre Anki → **Fichier → Importer** → sélectionne `flashcards/cards.tsv` :
- Type de note : **Basique**
- Séparateur : **Tabulation**
- Champ 1 → Recto · Champ 2 → Verso · Champ 3 → Tags

25 cartes sur Ch.1 sont déjà prêtes. Fais un premier tour rapide (5 min) — juste pour voir les questions, pas pour mémoriser.
